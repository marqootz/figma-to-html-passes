2.2 Frame Setup

  • Create a FRAME in Figma
  • Set the frame size to match your animation dimensions
  • Name it with the [LOTTIE] prefix
  • Position it where you want the animation to appear

üö® Troubleshooting

Animation Not Loading?

  • Check that the JSON file is valid Lottie format
  • Verify the filename in the frame name matches the actual file
  • Ensure you're using a web server (not opening file directly)
  • Check browser console for error messages

File Path Issues?

The system automatically tries these paths:

  • lottie/filename.json
  • ./lottie/filename.json
  • ../lottie/filename.json
  • /examples/lottie/filename.json
  • examples/lottie/filename.json
