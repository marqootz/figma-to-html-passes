How It Works

1
Create Video Frames in Figma

Name your frames with the [VIDEO] prefix followed by the video filename:

[VIDEO] demo-video.mp4 [VIDEO] presentation.webm [VIDEO] tutorial.mov
2
Export Your Design

When you export your design, the plugin will:

  • Detect all frames with [VIDEO] prefix
  • Extract the video filenames
  • Generate HTML with video elements
  • Download a setup guide with video file information
3
Setup Video Files

The plugin downloads two files:

  • Your HTML file - Contains the generated design
  • video-setup-info.json - Contains video file instructions
4
Create Video Directory Structure

Create this directory structure alongside your HTML file:

your-project/ ‚îú‚îÄ‚îÄ your-design.html ‚îú‚îÄ‚îÄ video-setup-info.json ‚îî‚îÄ‚îÄ video/ ‚îú‚îÄ‚îÄ demo-video.mp4 ‚îú‚îÄ‚îÄ presentation.webm ‚îî‚îÄ‚îÄ tutorial.mov
5
Place Video Files

Copy your actual video files into the video/ directory with the exact filenames specified in the setup guide.

Troubleshooting

Common Issues:
  • Video not loading: Make sure you're using a web server, not opening the file directly
  • Wrong filename: Check that video filenames match exactly (case-sensitive)
  • Video directory missing: Create the video/ directory alongside your HTML file
  • Unsupported format: Use MP4 (H.264) for best browser compatibility
