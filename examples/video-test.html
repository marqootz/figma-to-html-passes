Video Support Test - Figma to HTML

Video Support Features:

  • ✅ Automatic video frame detection with [VIDEO] prefix
  • ✅ Video filename extraction from frame names
  • ✅ HTML5 video element generation
  • ✅ Object-fit: contain for proper scaling
  • ✅ Data attributes for video identification
  • ✅ Fallback message for unsupported browsers

Example: Figma Frame with [VIDEO] prefix

In Figma, create a frame named: [VIDEO] test-video.mp4

The exporter will generate:

<div data-figma-name="[VIDEO] test-video.mp4" data-video-frame="true" data-video-filename="test-video.mp4">
  <video controls preload="metadata" style="width: 100%; height: 100%; object-fit: contain;">
    <source src="video/test-video.mp4" type="video/mp4">
    Your browser does not support the video tag.
  </video>
</div>

How to use video support:

  1. Place video files in the examples/video/ directory
  2. In Figma, create frames with names like [VIDEO] filename.mp4
  3. Export your design - video frames will automatically become video elements
  4. Serve your HTML with a web server that supports video MIME types
